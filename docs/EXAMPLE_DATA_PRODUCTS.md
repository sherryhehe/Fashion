## Usage Instructions

1. **First, add the brands** using the data from `EXAMPLE_DATA_BRANDS.md`
2. **Copy the JSON data** for each product you want to add
3. **Go to Admin Panel** → Products → Add Product
4. **Fill in the fields:**
   - Name: From the JSON
   - Slug: From the JSON (or auto-generate)
   - Description: Copy from JSON
   - Price: From the JSON
   - Original Price: From the JSON (for discount calculation)
   - Category: Select from dropdown (must match existing categories)
   - Brand: Select from dropdown (must match brands you added)
   - SKU: From the JSON
   - Stock: From the JSON
   - Images: Upload product images
   - Status: Set to "active"
   - Featured: Check if you want it featured
   - Variations: Add size and color variations
   - Specifications: Add from JSON

5. **Save the product**

## Important Notes

- **Category must exist first:** Make sure the category (e.g., "Dresses", "T-Shirts & Tops") exists in your database
- **Brand must exist first:** Add brands from `EXAMPLE_DATA_BRANDS.md` before adding products
- **Images:** Upload product images when adding (images array is empty in JSON)
- **Variations:** Add size and color variations as shown in the JSON
- **Pricing:** All prices are in Pakistani Rupees (PKR)
- **Stock:** Adjust stock levels based on your inventory

## Product Categories Used

- Dresses (most products)
- T-Shirts & Tops (some casual products)
- Shirts & Blouses (formal products)

Make sure these categories exist in your database before adding products!

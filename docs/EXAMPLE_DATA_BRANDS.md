## Usage Instructions

1. **Copy the JSON data** for each brand you want to add
2. **Go to Admin Panel** → Brands → Add Brand
3. **Fill in the fields:**
   - Name: From the JSON
   - Slug: From the JSON (or auto-generate)
   - Description: Copy from JSON
   - Logo: Upload brand logo (optional)
   - Website: From the JSON
   - Status: Set to "active"
   - Featured: Check if you want it featured
   - Popular: Check if you want it popular

4. **Save the brand**

## Notes

- All brands are set to `status: "active"` and `popular: true`
- Featured brands (Khaadi, Sana Safinaz, Maria B, Gul Ahmed, Nishat Linen) are marked with `featured: true`
- Logo URLs are empty - upload brand logos when adding
- Website URLs are provided for reference
- Product count starts at 0 and will update automatically when products are added
